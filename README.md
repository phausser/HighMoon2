# HighMoon2

A minimal TypeScript browser scene rendered with Canvas 2D.

## Stack
- TypeScript
- Browser Canvas 2D
- `tsc` build output to `dist/index.js`

## Run
```bash
npm run build
```
Then open `index.html` in your browser.

## Controls
- `ArrowLeft` / `ArrowRight`: rotate ship
- `ArrowUp` / `ArrowDown`: move ship vertically
